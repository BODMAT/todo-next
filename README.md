# To-Do

### Як запустити
1. Встановити залежності: `npm install`
2. Запустити в режимі розробки: `npm run dev`
3. Зібрати білд: `npm run build`

### Що реалізовано (по ТЗ + бонуси)
* **Весь базовий функціонал:** додавання, видалення та позначення тасок.
* **useReducer:** замість звичайних стейтів використав редьюсер для логіки (бонус).
* **Редагування:** можна змінити текст таски через подвійний клік (бонус).
* **Фільтрація:** додав фільтри (бонус).
* **LocalStorage:** все зберігається в браузері, дані не зникають після оновлення сторінки.
* **Custom Hook:** виніс усю логіку в `useTodos`, щоб компоненти були чистими.
